# Smiley

## 프로젝트명
반려견 행동 분류 및 감정 예측 모델을 활용한 양육일기 공유서비스

## 주요기능
- 영상 업로드시 반려동물 행동 및 감정 분석
- 분석된 감정으로 일기장 작성

## 개발환경
- 사용언어 : HTML / CSS / Javascript / Python / PyTorch / YOLO
- 개발도구 : Jupyter / VSCode 
- 서버환경 : nodeJS
- 데이터베이스 : Mysql
